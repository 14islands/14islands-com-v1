CSS Workshop - From Novice to Pro

What is it about?

This course is intended for novice and intermediate developers who already work with CSS in some capacity. It gives a practical understanding of the core CSS foundation as well as advanced CSS3 features that you can't live without.

We will study popular techniques for structuring CSS in large scale projects, using modularity to make your team more efficient.

Half day, 4h

  • CSS Foundation
  • Layouts & Positioning
  • Advanced CSS3 layouts
  • CSS3 Transitions, 2D/3D Transforms & Animations
  • CSS Pre-processors
  • CSS Architecture for big projects

Full day, 7h

In this extended course we build on what we learned in the previous workshop. We share clever techniques and useful strategies for building high-performing responsive web sites and applications.

  • All half day workshop included
  • Mobile First & Progressive Enhancement
  • Responsive layouts
  • Responsive images
  • Render Performance
  • Dev tools & debugging

Exercises

Both half day + full day includes exercises that can be performed individually or in small teams.
